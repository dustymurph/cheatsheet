HTML

Table tags

Tag (thead 1) Name (thead 2) Description (thead 3)
<table> Table Wrapper element for all HTML tables
<thead> Table Header Section Set of rows defining the column headers in a table
<th> Table Head Cell Individual header cells in a table and used within <tr>, content is bold and centred.
<tbody> Table Body Set of rows containing actual table data (like the meat of the table)
<tr> Table Row Row container
<td> Table Data Single element that holds data one at a time (like this!)
<tfoot> Table Foot Footer/of end of the table (for example, a 'Total amount' row)

Table attritbutes

Attribute Name Description
<colspan> Column Span Defines how many columns it should span for
<rowspan> Row Span Defines how many rows it should span for

Semantic Markup

Here's an example of how Semantic Markup works. Big shoutout to west221b, whom I took reference from, thank you!

<html lang="en">            
    <head>
        <title>Title</title>
    </head>
            
    <body>
            
        <header>
            Logo, title, navigation
            <nav>
                Primary and secondary menus but ever nest navs
            </nav>
        </header>
            
        <section>
            Related grouping of semantic meaning
            
        <article>
            Self-contained content
        </article>
            
        <article>
            Would still make sense if all other content removed
        
        <aside>
            <blockquote>
                Related to main but outside the main flow
            </blockquote>
        </aside>
            
            </article>
        </section>
            
        <footer>
            Copyright, contact, links
        </footer>
            
        </body>
            
    </html>
